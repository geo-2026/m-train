# 🚂 M-TRAIN — 자연과 인간의 관계 수업 활동 앱 (웹툰형)

중학교 도덕/사회 「자연과 인간의 관계」 단원을 위한 학생·교사 통합 수업 활동 앱입니다.
교사 PC에서 서버를 실행하면, 학생은 같은 와이파이(또는 공개 단축 URL)로 접속해 활동합니다.

---

## 1. 실행 방법 (교사용)

### 가장 쉬운 방법
폴더 안의 **`시작하기.bat`** 파일을 더블클릭하세요.
- 처음 한 번은 자동으로 필요한 파일을 설치합니다(인터넷 필요, 1회).
- 검은 창에 **학생 접속 주소**와 **QR 코드**가 표시됩니다.

### 공개 단축 URL로 열기 (학생이 다른 인터넷망일 때)
**`시작하기(공개URL).bat`** 를 더블클릭하면 `https://m-train.loca.lt` 형태의 공개 주소가 함께 생성됩니다. (인터넷 필요)

### 명령어로 실행
```bat
npm install      :: 최초 1회
npm start        :: LAN + QR
npm run tunnel   :: 공개 단축 URL 포함
```
포트를 바꾸려면: `set PORT=8090 && npm start`

---

## 2. 교실에서 사용하기

1. 교사가 위 방법으로 서버를 켭니다.
2. 검은 창(또는 앱 첫 화면의 **📱 학생 접속 주소 / QR 보기**)에 나온 **주소/QR**을 칠판·화면에 띄웁니다.
3. 학생은 휴대폰·태블릿·노트북으로 **같은 와이파이**에 접속한 뒤 QR을 스캔하거나 주소를 입력합니다.
4. 첫 화면에서 모두 **우리 반(1-1 ~ 1-6)** 을 고른 뒤
   - 학생 → **학생으로 시작**
   - 교사 → **교사로 시작**

---

## 3. 화면 구성 (페이지)

| 페이지 | 대상 | 내용 |
|---|---|---|
| 1 | 공통 | 학급(1-1~1-6) 선택 + 학생/교사 선택 |
| 2 | 학생 | 학번·이름 입력 |
| 3 | 교사 | Claude API 키 입력 → 바로 10페이지로 |
| 4 | 학생 | 산악열차 읽기 자료 + 성향 체크리스트(10문항) → 인간/생태 성향 자동 진단 |
| 5 | 학생 | 인간 중심주의 학습 |
| 6 | 학생 | 생태 중심주의 학습 |
| 7 | 학생 | O·X 형성평가 풀이 |
| 8 | 학생 | 정답 확인 (통과/재학습 안내) |
| 9 | 학생 | 칸트·레오폴드 소개 + 질문 작성 → **익명으로 DB 저장** → 12페이지로 |
| 10 | 교사 | 같은 반 학생 질문을 **DB에서 실시간 수집·표시**(익명). 질문 클릭 시 11페이지로 이동 → AI 대화하기 버튼 |
| 11 | 교사 | AI 칸트·레오폴드와 대화 + **10페이지로 돌아가 질문 확인** 버튼 + 대화 PDF 저장 |
| 12 | 학생 | 통합적 관점 읽기 + 내 질문 자동 표시 + 생각/느낀 점 입력 → 활동지 PDF 저장 |

### 형성평가(7·8페이지) 규칙
- **1차 도전**: 8문항 · 힌트 2회 · **5개 이상** 정답 시 통과 (미만이면 5·6p 재학습)
- **2차 도전**: 8문항 · 힌트 4회 · **5개 이상** 정답 시 통과
- **3차 도전**: 8문항 · 힌트 10회 · **2개 이상** 정답 시 통과
- 통과하지 못하면 학습 화면(5·6p)으로 안내 후 다음 단계 도전.

## 4. 저장되는 PDF

- **학생(12p)**: `학번_이름.pdf` — 성향 진단, 평가 결과, 사상가 질문, 통합적 관점 생각, 느낀 점.
- **교사(11p)**: `학급반 철학자 대화.pdf` (예: `1-3반 철학자 대화.pdf`) — 학생 **익명** 질문 목록 + AI 칸트/레오폴드 대화 전체.
- 한글은 글꼴(나눔고딕/맑은 고딕)을 PDF에 넣어 **깨지지 않게** 저장됩니다.
- PDF는 클릭한 **그 기기의 다운로드 폴더**에 저장됩니다.

---

## 5. 참고 / 문제 해결

- **API 키**: console.anthropic.com 에서 발급. 키는 **교사 브라우저에서만 잠깐 사용**되고(매 요청 시 전송) 서버·기기·깃 어디에도 저장되지 않습니다. 학생에게는 전송되지 않습니다. **앱을 열 때마다(새로고침 포함) 다시 입력**하며 언제든 변경 가능합니다.
- **AI가 응답하지 않아요**: 교사 화면의 *API 키 다시 입력* 버튼으로 키를 다시 입력하세요.
- **포트 충돌**: 다른 프로그램이 8080을 쓰면 `set PORT=8090 && npm start`.
- **데이터**: 수집된 질문/대화는 SQLite 데이터베이스 `data/mtrain.db`에 저장됩니다(서버 재시작에도 유지). 질문은 **익명**(학번·이름 없음)으로 저장되며, **학생의 학번·이름은 서버에 저장되지 않고** 본인 기기의 PDF로만 남습니다. 교사 API 키도 저장하지 않습니다.

원본 학습 자료: `DATA1`(산악열차), `DATA2`(관점·QUIZ), `DATA4`(사상가), `DATA5`(통합적 관점), `DATA6`(웹툰 이미지).
